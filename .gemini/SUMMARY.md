## 🎯 How to Use Your Application

### Quick Start (3 Steps):

1. **Start Backend**
```bash
cd server
npm run dev
```

2. **Start Frontend** (new terminal)
```bash
cd client
npm run dev
```

3. **Open Browser**
- Go to http://localhost:5173
- Register an account
- Start building your portfolio!

### Full User Flow:

1. **Register/Login** → Create your account
2. **Dashboard** → See overview
3. **Editor** → Build your portfolio
   - Fill Personal info
   - Add Experience
   - Add Education  
   - Add Skills
   - Add Projects
   - Add Social links
   - Configure Settings
4. **Save** → Click save button
5. **View** → See your public portfolio at `/u/username`
6. **Share** → Copy URL and share!
